# Guess the Number Game

This is a simple "Guess the Number" game written in Python.

## How to Play

1. The computer will think of a random number between 1 and 100.
2. You need to guess what that number is.
3. After each guess, the computer will tell you if your guess is too high or too low.
4. Keep guessing until you find the correct number.

## How to Run the Game

1. Make sure you have Python installed on your system.
2. Open a terminal or command prompt.
3. Navigate to the `guess_the_number` directory.
4. Run the following command:

   ```bash
   python game.py
   ```
